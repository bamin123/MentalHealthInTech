Introduction¶

Mental health has been a growing concern over the past few years however it has recently become an issue more present in the public eye. This tutorial is will demonstrate what changes are present in the mental health of individuals over the past two years.

We will be covering the following topics in this tutorial:¶

  • Installing the Libraries
  • Loading and Plotting the Data
  • Exploratory Data Analysis
  • Conclusion & Insight

Hypothesis¶

We will be looking at this data and determining if there is a relation between the mental health of an individual and their gender or the population size of their job or age.

Getting Started¶

First, download the 2014 dataset from https://www.kaggle.com/osmi/mental-health-in-tech-survey and the continued 2016 dataset from https://www.kaggle.com/osmi/mental-health-in-tech-2016. This dataset is in the form of a csv(comma-seperated values file) and contains a set of survey questions. We will look at the 2014 answers and the 2016 answers, find trends and then see how we can compare the two datasets. To read the dataset, we have to load our CSV file into a form that we can manipulate using python. To do this, we will use the pandas library. Pandas uses a datatype called a data frame, which puts the data into a table. FTo get a better understanding of pandas and dataframes, you can visit the following pages:

  • Documentation: http://pandas.pydata.org
  • Pandas Cheat Sheet: https://www.dataquest.io/blog/images/cheat-sheets/pandas-cheat-sheet.pdf

Tidying our data¶

Too much data!¶

We can see there are 62 questions and we only want to pick the ones we can use. So we're going to start of by making a new dataframe by selecting only certain column indicies. Make sure to use the .copy() when selecting indicies to ensure a deep copy of the dataframe is created! In addition, the primary key provided by the cvs reader does not have unique values at each row, so we need to create a new index.

Inconsistent data¶

The gender column and company size was an open ended question, so we need to create consistent data. We can do these all by iterating over the dataframe and editing the values.

What does this mean?¶

The survey asked individuals who work in computer science related fields questions about mental health. The survey population is 24% female, 74% male and .02% other. Although this is an intense difference, there are generally speaking far fewer women in the technoloy field than men. Since we can't compare men against women, lets consider if women in 2014 were any different than the women surveyed in 2016. We can repeat the same steps on the survey dataframe in 2014.

Summary¶

After exploring this dataset, we found that within the tech industry, there is a higher percentage within the women population that have mental health disorder, it is a 6 to 10 ration. For men, it is still 5 to 10. This data set demonstrates that the overall there is a high percentage of individuals that struggle with mental health disorders.

We found that when it came to age overall, the ages between 20-30 had the highest number of individuals who said to have mental health disorders.
